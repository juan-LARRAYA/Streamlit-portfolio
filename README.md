# 🌟 Portafolio Personal con Streamlit

¡Bienvenido a mi portafolio personal! Este proyecto ha sido creado utilizando [Streamlit](https://streamlit.io/) para mostrar mis habilidades, proyectos y experiencia profesional de una manera interactiva y atractiva.

## 🚀 Características

- **Página de inicio atractiva**: Introducción personalizada y una visión general de mi perfil.
- **Sección de proyectos**: Lista detallada de mis proyectos destacados, con imágenes, descripciones y enlaces a repositorios o demos.
- **Perfil profesional**: Resumen de mi formación académica, experiencia laboral y habilidades clave.
- **Contacto**: Formulario interactivo para que los visitantes puedan ponerse en contacto conmigo fácilmente.

## 🛠️ Tecnologías utilizadas

- **Streamlit**: Framework principal para el desarrollo de la interfaz web.
- **Python**: Lenguaje de programación utilizado para construir la lógica y los componentes.
- **HTML/CSS**: Personalización adicional para mejorar el diseño.



## 📦 Instalación

1. **Clonar este repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/portfolio-streamlit.git
   cd portfolio-streamlit
   ```

2. **Instalar dependencias**:
   Asegúrate de tener Python 3.7 o superior instalado. Luego, instala las dependencias requeridas:
   ```bash
   pip install -r requirements.txt
   ```

3. **Ejecutar la aplicación**:
   Inicia el servidor de Streamlit:
   ```bash
   streamlit run juan.py
   ```

4. Abre el navegador en [http://localhost:8501](http://localhost:8501) para visualizar la aplicación.
